# AI Website Auditor

A free, AI-powered website auditor built with React that generates comprehensive audit reports for any website. Reports include detailed analysis points, audit scores, and downloadable PDF exports.

## Features

- 🔍 **AI-Powered Analysis**: Uses Gemini AI to analyze website performance, SEO, accessibility, and security
- 📊 **Comprehensive Audit Report**: Evaluates multiple criteria including:
  - Performance metrics
  - SEO optimization
  - Accessibility compliance
  - Security best practices
- 🎯 **Audit Score**: Generates an overall score based on all evaluation points
- 📥 **PDF Download**: Export reports as downloadable PDF files
- 🚀 **Firebase Deployment**: Deployable on Firebase free tier
- 💰 **100% Free**: Uses only free APIs and services

## Tech Stack

- **Frontend**: React 19
- **Styling**: Tailwind CSS
- **PDF Generation**: jsPDF + html2canvas
- **AI Engine**: Google Gemini 3
- **Deployment**: Firebase Hosting

## Usage

1. Enter a website URL
2. Click "Analyze"
3. Wait for the AI to process the site
4. View the detailed report and score
5. Download the PDF report for your records
